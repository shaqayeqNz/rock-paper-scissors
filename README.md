# Rock Paper Scissors Game

This project is a Python-based implementation of the "Rock, Paper, Scissors" game. In this project, the user plays against the computer which chooses a random item between the choises. The game can continue until the user decides to quit.

## Project Structure
``` 
rock-paper-scissors/
|
|---src/
|   |---main.py - The main game executer
|
|---README.md - This file
```

## Requirements
- Python 3.13+

## Running The Project
to run this project:

```python
python src/main.py
```
The game can be playes by running `main.py` script.
